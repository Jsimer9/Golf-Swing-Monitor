I made this project for the Let There Be Speed Competition, so what better way to let there be speed than to measure it. For bat-and-ball style sports (Baseball, Golf, Cricket, etc), swing speed matters. Using Newton's equation F=ma, you can see that the faster the speed or acceleration of an object, the more force you can exert on the object. Knowing your swing speed can help you to make adjustments to your swing or form to create more efficient speed and exert more force. I made this project using an ESP32 and MPU6050 from Adafruit Industries. I used the Wifi capabilities of the ESP32 in order to have the monitor send speed data to a website, which you can view on your phone. I built this project for a total of $46.
